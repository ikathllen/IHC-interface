Guia sobre gestão ágil
de requisitos

Este guia é uma apresentação resumida de
práticas de gestão ágil de requisitos que atende
modelos de processos de qualidade em
desenvolvimento de software. Neste aspecto o
guia apresenta as etapas, métodos e práticas na
gestão de requisitos. Pesquisador: Luã Silva

Get started

Objetivos

  • Facilitar o entendimento de gestão ágil de requisitos
  • Apresentar ferramentas para gestão de requisitos
  • Definir etapas e processos na gestão de requisitos
  • Alinhar o gerenciamento de requisitos com modelos
    de qualidade de software
